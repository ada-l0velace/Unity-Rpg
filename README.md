# Setup Fabric 
(Linux)
Run on terminal
``` 
> sudo apt-get install python-pip
> pip install python
> fab setup
``` 
(Windows)
  run in this order:
``` 
1 - setupfabric.bat
2 - setup2.bat 
3 - setup3.bat
```
